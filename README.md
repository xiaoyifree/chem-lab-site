# Chem Lab Site

一个面向中学生的化学实验学习网站原型，采用独立仓库与独立目录管理。

## 当前目标

- 按初阶、中阶、高阶组织实验内容
- 为实验提供现象演示、反应信息、安全提示与课后练习
- 预留动画与音效资源目录，便于后续升级为更强交互版本

## 目录结构

- `index.html` 首页与实验入口
- `experiments/` 后续单实验页面目录
- `assets/images/` 图片素材
- `assets/audio/` 音效素材
- `assets/animations/` 动画素材
- `styles/` 样式文件
- `scripts/` 交互脚本
- `data/` 实验数据

## 本地运行

直接在浏览器打开 `index.html` 即可预览首版原型。
